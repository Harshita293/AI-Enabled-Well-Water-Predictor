# Dataset Description

## Dataset Overview
The dataset is used to predict well water availability based on environmental and historical parameters.

## Source
The data was collected from publicly available groundwater resources and observation records.

## Features
- Rainfall
- Water Level
- Season
- Year
- Location (if applicable)

## Preprocessing
- Missing values handled
- Irrelevant data removed
- Data normalized for machine learning models

## Files
- raw_data.csv : Original collected data
- cleaned_data.csv : Preprocessed dataset used for training
